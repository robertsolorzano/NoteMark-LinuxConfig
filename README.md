# NoteMark

A Note taking desktop app with out-of-the-box markdown support. Built with Electron and React. Configured styling to work for linux. (macos was default by creator)


## Project Setup

### Install

```bash
$ yarn
```

### Development

```bash
$ yarn dev
```

### Build

```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```
